## Detailed Competitor Analysis

### Glean
- **What it is**: Enterprise search + AI assistant
- **Strengths**: Great at finding documents, strong integrations
- **Weaknesses**: Very expensive ($50+/user/month), no workflow automation
- **Our advantage**: We DO workflows, not just search

### ServiceNow Virtual Agent
- **What it is**: Chatbot for ServiceNow ITSM
- **Strengths**: Deep ServiceNow integration, enterprise trust
- **Weaknesses**: Limited to ServiceNow ecosystem, rigid conversation flows
- **Our advantage**: We're platform-agnostic via MCP

### Moveworks
- **What it is**: AI copilot for IT and HR
- **Strengths**: Strong NLU, good automations
- **Weaknesses**: Black box AI, very expensive, SaaS only
- **Our advantage**: Open source, customizable, self-hostable

### Microsoft Copilot
- **What it is**: AI assistant across M365
- **Strengths**: Native M365 integration
- **Weaknesses**: Generic, not IT-specific, hallucination concerns
- **Our advantage**: IT-specialized, grounded with RAG
